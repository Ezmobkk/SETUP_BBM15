# SETUP_BBM15

Indicateur MetaTrader 5 pour detecter un setup ICT specifique et envoyer des alertes visuelles et sonores.

## Objectif

Ce projet vise a construire un indicateur MT5, pas un robot de trading automatique. L'indicateur servira uniquement a reperer un setup defini, puis a alerter le trader.

## Structure

- `docs/` : cahier des charges, definition du setup, journal des decisions.
- `src/` : code source MQL5 de l'indicateur.
- `tests/` : scenarios de verification et captures de test.
- `releases/` : versions livrables de l'indicateur.
