# Journal des decisions

Ce fichier garde la trace des decisions importantes prises pendant le projet.

## Decisions

- Le projet repart de zero.
- L'outil final sera un indicateur MetaTrader 5, pas un robot de trading automatique.
- Le trader garde la decision finale d'entrer ou non en position.
- Pour le setup bullish, l'order block M15 correspond a la derniere bougie baissiere, ou a la derniere sequence de bougies baissieres consecutives, juste avant l'impulsion haussiere qui cree le BISI/FVG.
